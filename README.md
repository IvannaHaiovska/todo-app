# Todo App

A sleek, responsive Todo List application built with HTML, CSS, and Vanilla JS.

## Demo
[Live Demo](https://marvelous-caramel-00ef85.netlify.app/)

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Filter tasks by All / Pending / Completed
- Task counters (Total, Completed, Pending)
- Persistent storage via localStorage
- Responsive design for mobile devices
- Accessible: keyboard navigation, ARIA labels

## Technical Notes
- Each task has a unique id and uses data-id in the DOM for linking JS logic
- Modal window for adding/editing tasks
- Only minimal comments where logic is not obvious (localStorage, toggleCompleted, renderTabs)
